# FoodLab Help

```
Welcome to FoodLab! Here's what you can do:

/add-profile       Set up dietary restrictions for a family member
/find-recipe       Search the web for a new recipe (e.g., /find-recipe Thai curry)
/weekly-menu       Generate a weekly meal plan + shopping list
/whats-for-dinner  Pick a recipe for tonight with ingredients + method
/what-can-i-make   Enter ingredients you have, get matching recipes
/rate              Rate a recipe 1-5 stars (e.g., /rate moroccan-tagine 5)
/foodlab-help      Show this help message

Every recipe has a version for each person's dietary profile.
Set up profiles first with /add-profile to get personalized results.

Browse the repo:
  recipes/mains/       All main dish recipes
  recipes/breakfast/   Breakfast recipes
  weeks/               Past meal plans + shopping lists
  reviews/ratings.md   All ratings
  profiles/            Dietary profiles
```
